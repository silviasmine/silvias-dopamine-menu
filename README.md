## running locally

git clone https://github.com/silviasmine/silvias-dopamine-menu.git
cd silvias-dopamine-menu
npm install
npm run dev

you'll need a supabase project and a .env.local file with your NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY.
